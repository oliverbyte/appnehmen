# Screenshots

Bitte Screenshots hier ablegen:

1. **startseite.png** - Screenshot von der Startseite
   - Zeige: Persönliche Begrüßung, "Dein Warum", Gewichts-Übersicht, Hauptbereiche (Mein Gewicht, Gewohnheiten, Heißhunger-Notfall)
   
2. **gewichtsverlauf.png** - Screenshot vom Gewichtsverlauf-Screen
   - Zeige: Statistik-Karten, Diagramm mit Verlaufskurve, Button "Neues Gewicht eintragen"
   
3. **gewohnheiten.png** - Screenshot vom Gewohnheiten-Screen
   - Zeige: Wochenübersicht mit farbigen Tagen, Gewohnheiten-Liste mit Checkboxen

## Empfohlene Größe
- Breite: 375px (iPhone-Breite) oder 400px
- Format: PNG mit transparentem oder weißem Hintergrund
- Qualität: Hoch, für gute Lesbarkeit

## Screenshots erstellen

1. App im Browser öffnen: `flutter run -d chrome`
2. Browser-Entwicklertools öffnen (F12)
3. Device Toolbar aktivieren (Strg+Shift+M / Cmd+Shift+M)
4. Gerät auswählen: "iPhone 12 Pro" oder ähnlich
5. Zu entsprechendem Screen navigieren
6. Screenshot erstellen (Browser-Tool oder Betriebssystem)
7. Hier als `gewichtsverlauf.png` und `gewohnheiten.png` speichern
